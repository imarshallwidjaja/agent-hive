# Agent Hive v1.5.0

This release adds Hive Builder, an OpenCode ad-hoc executor path for isolated changes that do not need the full feature/task workflow. It also ships the ad-hoc worktree tool surface and env-gated background delegation guidance used by primary agents.

## Added

- Added the packaged Hive Builder agent for inspect, isolate, execute, verify, commit, merge, and cleanup flows without creating Hive feature/task records.
- Added `hive_adhoc_worktree_create`, `hive_adhoc_worktree_commit`, `hive_adhoc_merge`, and `hive_adhoc_cleanup` to the OpenCode plugin tool contract.
- Added single-root and manifest-backed composite ad-hoc worktree support in `hive-core`.
- Added `background-delegation` guidance for sessions that enable OpenCode background subagents through `OPENCODE_EXPERIMENTAL_BACKGROUND_SUBAGENTS` or `OPENCODE_EXPERIMENTAL`.

## Changed

- Runtime agents now inherit base prompts before applying custom agent docs, preserving required workflow and safety rules for custom roles.
- OpenCode README, tool docs, data model docs, and generated agent guidance now document Hive Builder, ad-hoc handoffs, and background delegation boundaries.
- The Hive Builder registration path is simplified around the packaged agent contract.

## Fixed

- Generated skill materializations are preserved during custom skill loading.
- Ad-hoc tool handoffs now validate run IDs, workspace paths, registered branches, and composite manifests before downstream operations.
