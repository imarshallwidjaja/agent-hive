# Agent Hive v1.4.9

This fork release establishes the OpenCode-only npm path under `oc-arkive`.

## Changed

- Renamed the OpenCode npm package from `opencode-hive` to `oc-arkive`.
- Updated OpenCode install docs to use `"plugin": ["oc-arkive@latest"]`.
- Narrowed the GitHub release workflow to build/test `hive-core` and `oc-arkive`, publish only `oc-arkive` to npm, and create the GitHub Release.
- Updated release docs and release contract tests for the fork-owned package and recovery flow.

## Fixed

- Switched inactive workspace package references to local `hive-core` workspace dependencies so CI does not try to fetch unpublished `hive-core` versions from npm during `bun install`.
